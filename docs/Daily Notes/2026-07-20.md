- **A1 [HIGH] rate-limit `POST /auth/token`** ‚Äî added `_login_limit` dependency.
  Extended `rate_limit()` with an optional `bucket=` arg so `/auth/login` and

  non-`http(s)` scheme at the Pydantic boundary; `Automations.jsx` also guards
  the render (`/^https?:\/\//`) and uses `rel="noopener noreferrer"`.
- **[MEDIUM] transport headers** ‚Äî added `Strict-Transport-Security` (only when
  the request is HTTPS / a hosting domain is set ‚Äî never on plain-HTTP LAN) and a
  restrictive `Permissions-Policy`. CSP deliberately deferred (see below).

  0.0.9‚Üí0.0.18; upper bounds added to `anthropic`/`openai`/`docker`. Verified the

- **[HIGH] Infisical‚Üín8n.env fan-out** ‚Äî `write_n8n_env()` now writes only
  n8n-scoped keys (`N8N_` prefix + `N8N_ENV_ALLOWLIST`), not the whole vault.
