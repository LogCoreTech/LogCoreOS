# USERS.md

This file maps users to their personal folders. Ask at session start which user is active.

Each user has a dedicated folder under `USERS/` containing their profile, personal memory files, and tasks.

---

## How to Select a User

At session start:
1. Ask "Who am I helping today?" if it's not clear from context
2. Read `USERS/{name}/{name}_Profile.md`
3. Then read their `Long_Term_Memory.md` and `Short_Term_Memory.md`
4. Then load their `Tasks/tasks.json` and score for top 3

## Adding a New User

Use the LogCore App setup wizard, or manually:
1. Copy `USERS/_template/` to `USERS/{Full Name}/`
2. Fill in `{Full Name}_Profile.md` with their details and life priority order
3. Add them to the table above
4. Add them to MEMORY_MAP.md
